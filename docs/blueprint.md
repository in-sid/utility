# **App Name**: DrivePay

## Core Features:

- Input Form: Collect bill date, period (monthly, quarterly), start date of FY, bill number (optional), driver name, and signature/stamp uploads.
- Salary Calculation: Automatically calculate salary based on provided information and configured rules.
- Slip Generation: Generate salary slip PDF based on the calculations and input data.  Use AI tool to determine the most logical and aesthetically pleasing layout for each field.
- Download Slip: Allow the user to download the generated salary slip in PDF format.

## Style Guidelines:

- Primary color: Soft blue (#89B4FA) for a calm and professional feel, like clear skies or deep waters, associated with trust, stability, and reliability
- Background color: Light blue (#E6F0FA) for a clean, unobtrusive background
- Accent color: Pale violet (#B189FA) to complement the primary color, creating a balanced palette.
- Body and headline font: 'Inter' sans-serif for a clean and modern look.
- Clean and organized layout with clear sections for input fields and generated slip preview.
- Use simple and recognizable icons for actions like 'download' and 'upload'.
